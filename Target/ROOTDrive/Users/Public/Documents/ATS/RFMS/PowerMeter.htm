Power Meter Help

 

Form Controls

This section explains the various controls of the Power Meter.

 

Controls

There are several controls available to setup and take measurements using the power meter. Some of these controls are required to take accurate measurements, others are optional.

 

Power Meter On/Off

This control is required because the power meter must be turned off before power heads can be attached or detached. The power meter must be turned on via this control before measurements can be taken.

 

Expected RF Frequency

Type a frequency into this edit box, then select one of the dimensional unit buttons from the list that appears to the right of this control, or press <enter> to use the current units.

 

Range (Expected RF Level)

The range control allows specification of a minimum expected power level (Lower), a maximum expected power level (Upper) or an auto detection mode (Auto). When Upper or Lower is selected, an edit box appears to the right of this control in which an appropriate power level must be entered, and dimensional units buttons must be selected from the list that appears to the right of this control, or press <enter> to use the current units.

 

Power Head (table)

Use this control to select either the power head being used or a custom table from which frequency dependent correction factors can be obtained.

 

Power Correction Factor

This control allows the manual entry of a frequency dependent correction factor. The entered correction factor is only used when NONE is selected in the Power Head (table) combo box. Once an entry is made in this edit box, one of the dimensional unit buttons from the list that appears to the right of this control must be selected, or <enter> can be pressed to use the current units.

 

Power Offset

This control allows the entry of an offset to the measurement. It may be used to compensate for any additional amplification or attenuation at the input for the power meter. Once an entry is made in this edit box, one of the dimensional units buttons from the list that appears to the right of this control must be selected, or <enter> can be pressed to use the current units.

 

Number Of Averages

This control is used to enter the number of time times to average a measurement.

 

 

Measurement Modes

Two measurement modes are supported, relative and absolute.

 

Absolute Mode

Select the ABS button to take absolute measurements. Absolute measurements are measurements with respect to ground. The dimensional units available in this mode are dBm, W, and volts.

 

Relative Mode

Select the REL button to take relative measurements. Relative measurements are measurements with respect to the last absolute measurement taken. The dimensional units available in this mode are percent and dB.

Measurements

Measurement results appear in the measurement frame with the dimensional units just to the right. To change the dimensional units click on the currently displayed units. Every time this control is click on it the next available units become visible and is selected.

 

 

Zeroing and Calibration the Power Meter

It is critical to first zero and calibrate the power meter before taking measurements. To zero and calibrate the power meter, attach the power sensor head to the power reference, being careful to add the attenuator for any low power heads. Select Commands->Zero + Cal from the menu bar. The measurement display indicates that a zero or calibration operation is in progress and a pop-up message is displayed when the procedure completes.

 

Enabling the Calibration Reference

The calibration reference can be enabled at any time by selecting Commands->Enable Cal Reference from the menu bar.
