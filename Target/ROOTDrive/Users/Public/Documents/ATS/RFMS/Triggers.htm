Trigger Modes

 

To view the trigger dialog box select, View->Trigger Menu from the menu bar. The Trigger Control dialog box has many trigger options that are described below.

 

Trigger Source                                                                                                      

The trigger source field contains four options.

 

Immediate

In Immediate mode the instrument begins capturing data for the measurement as soon as the Measure button is pressed.

 

External

In external mode the instrument begins capturing data for the measurement as soon as the signal connected to the External Trigger input satisfies the conditions set by the slope field and the trigger level.

 

Input mode

In Input mode the instrument begins capturing data for the measurement as soon as the input signal (signal being measured) meets the conditions set by the slope field and the trigger level.

 

TTL

In TTL mode the instrument begins capturing data for the measurement when the signal on the selected TTL backplane pin satisfies the conditions set by the slope field and the trigger level.

 

TTL Bk Plane

This field set which TTL back plane trigger input is to be used. It is only visible when TTL is selected from the Trigger Source field.

 

Trigger Slope

The trigger slope field sets the trigger event to be dependant on either a rising or falling edge of the trigger source. This field is disabled when the source is Immediate.

 

Trig Delay

Sets period of time that must pass before a measurement is taken.

 

Trig Level

This input field sets the level at which the trigger source signal triggers. It is only enabled when the input Trigger Source is External.
