Using the Table Editor

 

The Table Editor is used to manage Power Head tables. To launch the table editor from the menu bar select View->Table Editor.  The following dialog box will open.

 

 

Open the Model # combo box in the Power Head frame to select a power head table to edit.

 

Editing the Serial Number

The serial number of the Power Head is displayed in the Serial #: edit box. To change it, click the Edit Serial # button, make the changes, then click the Save button in the Power Head Frame.

 

Editing the Attenuator Serial Number

The Attenuator serial number of the Power Head is displayed in the Serial #: edit box. Not all of the power heads require an attenuator, and this feature is only available for those that do. To change it, click the Edit Serial # button, make the changes, then click the Save button in the Power Head Frame.

 

Editing the Reference Correction Factor

To change the reference correction factor for the power head click the Edit RCF button to enable the Reference CF edit box. Type the correction factor in the Reference CF % edit box and click the SaveRCF button.

 

Editing Correction Factors

To edit existing correction factors use the drop down Freq (MHz) combo box to select the frequency. Enter the new correction factor in the CF % edit box and click the Add button. Although you can click the Save button in the Correction Factors frame anytime to save your changes, it is only necessary to do so when all of the changes are made.

 

Adding Correction Factors

To add a correction factor, enter the desired frequency for the correction factor in the Freq (Mhz) combo box. Enter the correction factor in the CF % edit box and click the Add button. Although you can click the Save button in the Correction Factors frame anytime to save your changes, it is only necessary to do so when all of the changes are made.

 

Deleting Correction Factors

To delete correction factors, use the Freq (Mhz) combo box to select the frequency that is to be deleted, and click the delete button. Click the Add button. Although you can click the Save button in the Correction Factors frame anytime to save your changes, it is only necessary to do so when all of the changes are made.
