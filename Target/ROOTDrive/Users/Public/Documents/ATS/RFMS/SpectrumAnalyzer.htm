RF Spectrum Analyzer Help

 

Form Controls

This section explains the various controls of the spectrum analyzer.

 

Frequency Controls

The frequency button, located just left of the spectrum graph, must be pressed to view, select and change any of the frequency controls. The frequency controls include:

 

·        Center Frequency

·        Span

·        Start Frequency

·        Stop Frequency

·        Resolution bandwidth

·        Frequency windows

 

Clicking on any of these frequency control buttons allows you to change the current instrument setting. Once a button is clicked, an edit box is opened on the left center of the graph. This edit box displays the instrument setting description and its current value. Notice that another set of buttons becomes visible just left of the frequency buttons. These buttons allow you to select the measurement units that are associated with the number you enter in the edit box. It also allows you to cancel the operation, and in some cases, select auto modes. Most of the other controls on the form will be disabled until one of these buttons is selected.

 

Amplitude Controls

The amplitude button, located just left of the spectrum graph, must be pressed to view, select and change any of the amplitude controls. The amplitude controls include:

 

·        Reference Level

·        Attenuator Level

·        Peak Hold

·        Averaging

·        dB/Div

 

Clicking on any of these amplitude control buttons allows you to change the current instrument setting. Once a button is clicked an edit box is opened on the left center of the graph. This edit box displays the instrument setting description and its current value. Notice that another set of buttons becomes visible just left of the amplitude buttons. These buttons allow you to select the measurement units that are associated with the number you enter in the edit box. It also allows you to cancel the operation, and in some cases, select auto modes. Most of the other controls on the form will be disabled until one of these buttons is selected.

 

Markers

To enable and use markers, first click the check boxes in the Markers frame. Select either one or both of the available markers. Then press the Peak Search button(s) and press the Measure button. The markers will appear on the graph at the highest point, and a markers section appears in the upper left corner of the graph. The markers can be clicked and dragged along the waveform. The arrow buttons can also be used to move the markers from one peak to the next.

 

To compare the markers, click the Display Difference check box in the Markers frame. When this is done the blue marker information on the graph display the frequency and power difference between the markers.

 

To set a marker at a specific frequency, press the Markers button. Two new buttons become visible below the Markers button. Use these buttons to place a marker at the desired frequency.

 

Display Features

 

Resize

The Spectrum Analyzer form can be resized like any windows form. Use this feature to increase the default size of the graph. The default size is the minimum size.

 

Colors

The Spectrum Analyzer graph supports several color combinations. To view and select one of the color schemes, select View->Preview/Select Graph Colors from the menu bar. A text box will appear in the upper right side of the graph instructing you to click on the box to select the current color scheme. The colors schemes will cycle through the available options until you click inside the box. When running in continuous mode the color schemes update very slowly, so you may want to pause to view the choices.

 

Plot Features

The following plotting features are supported:

 

·        Plot to file (save current displayed graph to a file)

·        Plot from file (displays a graph from a file)

·        Add from file (adds file data to the currently displayed graph)

·        Sub from file (subtracts file data from the currently displayed graph)

 

These plot feature are available by selecting Plot from the menu bar.

 

Vertical Scale

Two types of vertical scales are available; linear and logarithmic. To change the current vertical scale, select Plot->Vertical Scale from the menu bar.

 

 

Yig Tuned Filter

To enable the yig tuned filter, select Commands->Yig Filter from the menu bar. A check appears next to the yig filter when this option is enabled. To disable it, click in it again.
