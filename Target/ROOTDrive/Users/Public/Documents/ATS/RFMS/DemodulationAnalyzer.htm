Demodulation Analyzer Help

 

Form Controls

This section explains the various controls of the demodulation analyzer.

 

Expected Signal Characteristics Controls

The three controls on the left side of the form are used to configure the Demodulation Analyzer to lock onto the signal to be measured and enable it to collect sufficient data samples to take the desired measurement.

 

Min Mod Frequency (Minimum Expected Modulation Frequency)

It is important to enter the minimum expected modulation frequency via this control. This information is necessary so that the instrument collects enough data points to accurately represent the signal that is going to be evaluated.

 

There are several ways to use this control.

·        Expand the combo box and select one of the items, then select one of the dimensional units buttons from the list that appears to the right of this control.

·        Type a frequency into this combo box, then select one of the dimensional units buttons from the list that appears to the right of this control.

·        Expand the combo box and select Auto for auto detection of the expected modulation frequency.

 

Exp Carrier Freq  (Expected Carrier Frequency)

Type a frequency into this combo box, then select one of the dimensional units buttons from the list that appears to the right of this control.

 

Exp Input Power (Expected Carrier Input Power)

Type the expected carrier input power level into this combo box, then select one of the dimensional unit buttons from the list that appears to the right of this control. The instrument has automatic gain control and may change this setting automatically.

 

Modulation

The modulation frame contains buttons for selecting the expected type of modulation. This control also updates the several other controls on the form. When a modulation button is selected the measurement type combo box, and the measurement units combo box in the Measurement frame update with valid selections for the signal type. The Post Detection Filter combo boxes also update with only valid selections for the given signal type. Here is a list of the modulation signal types supported.

 

·        AM (Amplitude Modulation)

·        FM (Frequency Modulation)

·        PM (Phase Modulation)

·        PAM (Pulse Amplitude Modulation)

·        PAC (Pulsed Amplitude Modulation)

 

Post Detection Filters

Several post detection filters are available. Not all filters are available for all type of modulated signals. These combo boxes update with only the valid choices when a modulation type is selected.

Carrier Correction

Carrier correction corrects for inaccuracies due to phase differences between the signal source and the measurement system. It is a good idea to select the Carrier Correction check box when the signal source and measurement system are not sharing a common reference source. Not using this feature can lead to invalid measurements.

Measurements

The Measurement frame contains many controls:

 

Measurement Type

The upper left most combo box is used to select the type of measurement.

 

Measurement Units

The upper right most combo box allows the selection of the measurement units to be used.

 

Taking the measurement

Once all other instrument settings and measurement settings have been made there are two buttons used to make measurements.

 

·        Capture + Demod

·        Fetch Measurement

 

Pressing the Capture + Demod button causes the instrument to collect, process and store a waveform in memory. This waveform is also displayed in the time domain graph. The Fetch Measurement button performs measurements by evaluating this waveform that is stored in memory. The results are displayed in the center text box in the measurement frame. If you want to take a new measurement on new waveform, click on the Capture + Demod button again. In some cases, such as taking Modulation frequency measurements, this method of doing a Capture + Demod, and evaluating the waveform stored in memory, is not available and only the Fetch Measurement button can be used to obtain the measurement since the Capture + Demod button is disabled.

 

Display Features

 

Resize

The Demodulation Analyzer form can be resized like any windows form. Use this feature to increase the default size of the graph. The default size is the minimum size.

 

Colors

The Demodulation Analyzer graph supports several color combinations. To view and select one of the color schemes, select View->Preview/Select Graph Colors from the menu bar. A text box will appear in the upper right side of the graph instructing you to click on the box to select the current color scheme. The colors schemes will cycle through the available options until you click inside the box. When running in continuous mode the color schemes update very slowly, so you may want to pause to view the choices.

 

Plot Features

The following plotting features are supported:

 

·        Plot to file (save current displayed graph to a file)

·        Plot from file (displays a graph from a file)

·        Add from file (adds file data to the currently displayed graph)

·        Sub from file (subtracts file data from the currently displayed graph)

 

These plot feature are available by selecting Plot from the menu bar.

 

Audio Features

The instrument is capable of demodulating audio signals and saving them to a file in a WAV file format. It can also play these files back through the PC speaker. Any audio waveform currently in memory can be saved by selecting File->Export to Wav from the menu bar. These files can be opened and played back by selecting File->Play Wav File from the menu bar. Due to the hardware and memory limitations, only 3 seconds of auto data can be collected.
