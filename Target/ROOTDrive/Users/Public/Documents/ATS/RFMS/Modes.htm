Modes

 

To view the instrument modes click the Modes selection on the menu bar.

 

Interactive Mode

Interactive mode enables the user to interact with and control the instrument. In this mode the user has complete control over the instrument and is free to changes any of the settings and take measurements. When the instrument is in Interactive mode there is a check mark next to Interactive. Interactive mode is paired with Monitor mode, such that clicking Monitor mode disables Interactive mode.

 

Monitor Mode

Monitor mode is a spy mode that enables the user to monitor or spy on the state of the instrument as it is being controlled by another user. Monitor mode disables most instrument features to ensure that no settings are altered. Monitor mode is included to aid in debugging applications that are using the instrument. When Monitor mode is enabled there is a check mark next to Monitor. Monitor mode is paired with Interactive mode, such that clicking Interactive mode disables Monitor mode.

 

Simulation

Simulation allows the soft front panel to operate without any instrument hardware. In this mode everything is simulated. It is intended for training and development. To enter or exit Simulation mode click Simulation. Simulation mode is on when there is a check mark next to Simulation. This soft front panel enters Simulation mode automatically when it detects any hardware errors on startup.

 

Continuous

In Continuous mode the instrument measures continuously and the Measure button’s text changes to “Pause”. Pressing this “Pause” button pauses the measurements, but does not exit Continuous mode. Clicking on Continuous toggles this mode on and off. The instrument also exits Continuous mode anytime it encounters an error.

 

Reset On Exit

When Reset On Exit is checked the instrument resets the soft front panel when it closes. Clicking Rest On Exit toggles this mode on and off.

 

Lock

When Lock is checked the instrument is no longer available to other users. Any other user of this instrument is locked out and receives an error message when an attempt is made to alter any settings or take a measurement. This feature enables a user to maintain complete uninterrupted control of the instrument for as long as the Lock is active. It is recommended that this mode be used as little as possible as it causes other users of the system to receive errors. Clicking Lock toggles the Lock mode on and off.
