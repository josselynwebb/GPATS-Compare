RF Frequency Counter Help

                                                      

Form Controls

This section explains the various controls of the RF Frequency Counter.

 

Expected Signal Characteristics Controls

Three controls in the center of the form are used to configure the RF Counter to lock onto the signal to be measured and enable it to collect sufficient data samples to take the desired measurement.

 

Expected RF Frequency

Type a frequency into this edit box, then select one of the dimensional units buttons from the list that appears to the right of this control, or press <enter> to use the current units.

 

Expected RF Level

Type the expected carrier input power level into this edit box, then select one of the dimensional unit buttons from the list that appears to the right of this control, or press <enter> to use the current unit. The instrument has automatic gain control and can change this setting automatically.

 

Upper Pulse Frequency Range

It is important to enter the minimum expected pulse frequency via this control. This information is necessary so that the instrument collects enough data points to accurately represent the signal that is going to be evaluated.

 

There are two ways to use this control:

·        Expand the combo box and select one of the items.

·        Expand the combo box and select Auto for auto detection of the expected pulse frequency.

 

Measurement Modes

The measurement modes form contains buttons used to select the type of measurement. Click on the button that corresponds to the type of measurement being taken. The following measurement types are supported.

 

·        Pulse Period (Pulse Frequency)

·        Pulse Width

·        RF Frequency (Carrier Frequency)

·        Peak Search

Measurements

Measurement results appear in the measurement frame with the dimensional units just to the right. To change the dimensional units click on the currently displayed units. Every time this control is selected the next available units become visible and are selected.
