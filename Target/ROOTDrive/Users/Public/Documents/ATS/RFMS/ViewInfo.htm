Viewing the Trigger Menu

 

 

Viewing Hardware Information

 

To view hardware serial number, versions and driver version information, click View->RFMS Hardware from the menu bar. A dialog box pops up displaying all of the available information.

 

 

Viewing Status Information

 

To view the instruments status information select View->Status Info from the menu bar. The bottom of the screen drops down and displays the fields listed below. Clicking Status Info toggles this view on and off.

 

Open

This field indicates if a session to the instrument is open.

 

Lock

This field indicates if the instrument is locked.

 

Own Lock

This field indicates if this user owns the lock.

 

Clients

This field indicates the number of clients accessing the instrument.

 

Simulation

This field indicates if the instrument is in simulation mode.
